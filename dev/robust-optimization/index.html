Robust Optimization

For an overview of different robust optimization models, we recommend the paper [1]. As a general reference for fundamental mathematical concepts and notation such as propositional logic and sets, we recommend the book Logic and Proof [2]. For model building using convex optimization, we recommend MOSEK Modeling Cookbook [3].

We denote scalar using lower-case math italics, vectors using lower-case boldface symbols, and sets using upper-case symbols. All non-matrix algebra operations on vectors are element-wise.

We denote vector constructors using $(element∣condition)$ and set constructors using $\{element∣condition\}.$

  • 2Avigad, J., Lewis, R., & van Doorn, F. (2020). Logic and Proof. Retrieved from https://leanprover.github.io/logic_and_proof/
